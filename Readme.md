## Create a virtual environment
```bash
python3 -m venv venv
```

## Activate the virtual environment
```bash
source venv/bin/activate
```

## Install dependencies
```bash
pip install -r requirements.txt
```

## Run the service
```bash
python __main__.py
```

## Deactivate the virtual environment
```bash
deactivate
```
